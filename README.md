# excel2design

将 Excel 数据转换为设计文件（架构图、Excalidraw 图等）的工具集。

## 目录结构

```
excel2design/
├── README.md
├── .gitignore
├── src/                # 源代码
├── input/              # 输入的 Excel 文件
├── output/             # 生成的设计文件
├── tests/              # 测试代码
└── docs/               # 文档
```
